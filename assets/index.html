Method of setting up widget
  1. It is a long push in the Home screen.Or, Please choose [menu button] -> [Add].
  2. "Add to Home screen" Because the screen is displayed, Please choose "Widgets".
  3. "Choose widget" Because the screen is displayed, Please choose "SwitchWallpaper".

How to use
    When the widget is arranged in Home, the clock is renewed at intervals of the second.
    A design different in each widget to which the clock is renewed in the character set at the end cannot be updated.
    When the TaskKiller application program is used, this application program might be ended. Please set not to be ended to "Ignore".

Help
    Please relocate the widget when operation becomes amusing.
    Still, please reinstall the application program when it is amusing.
